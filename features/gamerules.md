# Gamerules
LlamaBlocks has several gamerules which are used to configure various parts of its behavior. Below is a table of them all:

| Technical name              | Display name                              | Description                                                                                                                                  | Type    | Default |
|:----------------------------|:------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------|:--------|:--------|
| `allowItemDeletion`         | Allow Item Deletion Buttons to Function   | This controls whether the "Delete Items" button in the Netherite Backpack and the "Clear" button in the Secure Storage Block work.           | `logic` | `true`  |
| `giveAllRecipes`            | Give Players All Recipes After Joining    | This controls whether all players are given all crafting recipes when they join.                                                             | `logic` | `true`  |
| `giveGoldenBananaEffects`   | Enable Golden Banana Effects              | This controls whether the players get effects from eating a Golden Banana.                                                                   | `logic` | `true`  |
| `preventNestedStorageItems` | Prevent Placement of Nested Storage Items | This controls whether LlamaBlocks stops Shulker Boxes, Bundles, Backpacks, etc, from being placed in the Secure Storage Block and backpacks. | `logic` | `true`  |

# 1.1.2
[Released](https://github.com/Omegabird113/llamamod/releases/tag/1.1.2) January 24th, 2026
- Added support for the numbers `pi` and `e` in the computer calculator
- Fixed a bug where the link sent when you run `/llamablocks about` was not clickable
- Changed the link in about from GitHub to Modrinth
- Added the Configuration Files MCreator plugin to credits.
- Renamed the server configuration file from `llamamod_server.toml` to `llamablocks.server.toml`
- Added a bunch of 1-value operations to the computer: `NLOG`, `ROUND`, `CEIL`, `FLOOR`, `SQRT`, `3ROOT`, `ABS`, `SIGNUM`, `SIN`, `COS`, `TAN`, `ASIN`, `ACOS`, `ATAN`, `RAD>DEG`, and `DEG>RAD`
- Removed several parts of the `/lamablocks` command deemed unnecessary
- Added recipes for Tile Trapdoors
- The Secure Storage Block recipe now always requires a Netherite backpack

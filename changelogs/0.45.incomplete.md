# 0.45
Released January 1st, 2026
> <ins>**NOTE:**</ins> This is the first update on GitHub, so the changelog likely is not fully complete.

## Changes
- Added Large Tiles w/ various color variants
- Added Trapdoor variants of all building blocks (bricks, large tiles, & tiles)
- Added Teal Bricks & Large Tiles + variants
- Re-textured the Secure Storage Block, and Authenticator Block.
- Added the Variable Light Block
- Removed the About GUI, and replaced it with a message that appears in chat when you run "/llamamod about"
- Removed the "/llamamodabout" command
- Added Tile & Ceiling Tile fences & fence gates
- Updated the Time string in the computer to be more clear & detailed (Ex. "~7 (10% through the day/night)" would now be "7 o'clock (10% through the day)")
- Added/Updated/Removed tons of crafting recipes
- Re-added giving player all recipes automatically when they join a world.
- Added the ability to make the computer give a redstone output, including a random one.
- When an operator attempts to kill or clear the inventory of a player that does not exist in the Computer, it now gives an error.
- The error for invalid usernames in the computer now reminds the player that the name field is case-sensitive
- There are now check boxes which control whether or not the "Delete Items" button is visible in the Netherite backpack and Secure Storage Block GUIs, in order to prevent accidental item deletions
- The Creative Tab Icon was changed to the Lime Large Tile block
- Probably some other things I forgot

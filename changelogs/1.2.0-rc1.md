# 1.2.0 Release Candidate 1
- Updated LlamaBlocks Banner Pattern registry name from `llamablocksbannerpattern` to `llamablocks_banner_pattern` (**This breaks existing Banners from Beta 3**)
- The Golden Banana's saturation value has been decreased from `12.2` to `7.1`
- The Golden Banana now give Regeneration 1 for `0.7 seconds` and Luck 1 for `45 seconds` when eaten
- The Time in the Computer GUI and in `/llamablocks misc send time` now includes minutes, it's more accurate, and the percentage through the day/night is now formatted to the tenths place.
- Added a shapeless crafting recipe where `3 Farm Scraps` + `1 Milk Bucket` + `1 Honey Bottle` + `1 Poisonous Potato` = `1 Potato`
- Improved the chat Beta warning so it appears last in `/llamablocks about` and includes a GitHub link
- Made internal code improvements and updated texture names
- The password argument of `/llamamod password set` is now quotable
- Renamed the `password_control_passwords_permission_level` config option to `password_control_permission_level`
- There are now Villager trades for Bananas and Farm Scraps
- Re-arranged the textures on each side of the Variable Light
- `/llamablocks misc send time` now send the text in Green
- Reduced Banana saturation from `4.1` to `3.65`
- Updated the description of LlamaBlocks in the NeoForge about

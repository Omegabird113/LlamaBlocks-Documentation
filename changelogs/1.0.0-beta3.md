# 1.0.0 Beta 3
[Released](https://github.com/Omegabird113/llamamod/releases/tag/1.0.0-beta3) January 3rd, 2026
- Renamed the `Acid Disolving` effect to `Acid Dissolving` and did the same to its registry name.
- Added the `Acid Lake` feature which generates in the outer end islands
- Players who come in contact with Acid now receive the effect for `150 ticks` instead of `165 ticks`
- Changed affect the effect tick damage formula from `(a + 0.87) * 1.15` to `(a + 1.05) * 1.18`
- Added damage type `acid_drowning`
- Changed the acid drowning damage from type `drowning` to type `acid_drowning`
- Added a configuration file named `llamamod_server.toml` with the following config values: `generate_acid_lakes`, `access_existing_passwords_permission_level`, and `give_all_recipes_on_player_join`
- Updated the backpack recipe to replace its paper with any wooden plank
- Decreased the success rate for putting Farm Scraps in a composter from `17.5%` to `15%`
- Added a link to the GitHub repository to the message that is sent when a player enters `/llamamod about`
- Made the title of the Incorrect Password GUI a brighter red

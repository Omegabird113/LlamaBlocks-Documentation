# 1.3.0 Beta 3
[Released](https://github.com/Omegabird113/llamamod/releases/tag/1.3.0-beta3%2Bmc1.21.8) March 8th, 2026

- Added Cracked Large Tile variants
- Numbers can now be set to `RAND`, `RANDSTD`, `INF`, `-INF`, and `NAN` in the Computer Calculator
- Added an Automatic checkbox to the Computer Calculator
- Added more varieties of crafting recipes for the Cracked Sets
- The LlamaBlocks creative tab's icon is now the `Cracked Lime Large Tile`
- Updated the Incorrect Password Screen to look better & for its text to be more clear
- Updated the texture of the `Industrial Bricks` and `Industrial Large Tile` to be more consistent in terms of coloring with the Cracked variants
- Updated the beta warning message to have better grammar & be localizable
- Internal code improvements

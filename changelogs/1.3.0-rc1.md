# 1.3.0 Release Candidate 1
[Released](https://github.com/Omegabird113/llamamod/releases/tag/1.3.0-rc1%2Bmc1.21.8) March 29th, 2026

## Changes
- The Fences & Fence Gates of Cracked block variants now have the cracks lifted higher in their textures
- Changed the java package name to `io.github.omegabird113.llamablocks`
- LlamaBlocks now has its own dedicated Markdown documentation
- Internal code improvements

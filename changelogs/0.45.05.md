# 0.45.05
[Released](https://github.com/Omegabird113/llamamod/releases/tag/0.45.05) January 2nd, 2026
> **WARNING**: The mod-id and the registry names of almost every item were all changed. This means any world from previous LlamaMod versions will have **all LlamaMod Blocks/items disappear**.
 
## Changes
- The mod id is now "llamamod", not "llama_mod"
- Item & block registry IDs now have underscores
- Stair items are now pluralized in GUis

# 1.2.0 Beta 1
[Released](https://github.com/Omegabird113/llamamod/releases/tag/1.2.0-beta1) January 28th, 2026

## Changes
- The Banana texture has been rotated
- Added the stronger Golden Banana crafted when a banana is surrounded by gold with a brighter texture
- The warning in the Password Changer that operator's can see your passwords is now its own text instead of a tooltip
- Added the 1 value operator `log10` to the Computer Calculator
- Improved the code of the Computer Calculator
- Updated the Map colors for Ceiling Tiles and non-decorative blocks
- Added a "BETA" text in the Computer GUI which will be enabled in beta versions

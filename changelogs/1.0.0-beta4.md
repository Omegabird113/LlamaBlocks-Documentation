# 1.0.0 Beta 4
Released January 10th, 2026
- Banana Plants now have dedicated `Banana Patch` feature to generate them
- Banana Plants now only generate in the following biomes: `Flower Forest`, `Forest`, `Jungle`, `Cherry Grove`, `Plains`
- Updated `llamamod_server.toml` to add the following options: `generate_banana_patches`, `allow_features_to_delete_items`, and `prevent_nested_storage_item_placement`
-  The `allowItemClearing` gamerule was removed and replaced with  the `allow_features_to_delete_items` config option
- The LlamaMod Logo has been changed to make it more saturated
-  The LlamaMod logo in the mods menu is now scaled up to a higher resolution
- The Banana Plant texture has been changed to make them look better
- Banana Plants are now tinted as `Birch Foliage`
- You can no longer place Shulker Boxes or backpacks in a backpack or Secure Storage Block, though this limitation can be disabled in the server config
- The title text of the Backpack GUI is now brown
- Changed the names of a ton of textures to removes names like `new_<blockname>` and `<itemname>_2`
